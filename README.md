-Description-
Version ijava de Wordle,
Trouvez des mot de 5 lettres en 6 essais pour avoir le meilleur score.
*Mode Histoire et Mode Arcade Disponibles(voir Règles in-game pour plus d'informations)*
Chaque mot trouvé rapporte de 1 à 6 points en fonction du nombre d'essais, enchainez les mots pour augmenter votre score.
Enregistrez vos score,regardez les Règles ,le tableau des scores et la liste de mots.
--------------------------------------------------------------------------
-Statut-
Version Beta
--------------------------------------------------------------------------

-Lancement du jeu-
Afin d'utiliser le jeu, il doit être suffisant de taper les 
commandes suivantes depuis le répertoire src :
ijava compile Nerdle.java
//compilation des fichiers
ijava execute Nerdle
//lancement du jeu
--------------------------------------------------------------------------

-Menu-
    -1 pour Jouer,
    -2 pour le Tableau des Scores,
    -3 pour les Règles,
    -4 pour le Dictionnaire,
    -5 pour Quitter,
(et 6 pour lancer une version debug de Nerdle qui donne le mot au départ)
--------------------------------------------------------------------------

-Notes-
Pour sortir des Scores,Règles,et Dictionnaire Appuyer sur Entrée
Pour sortir de Nerdle Appuyer sur n'importe quelle touche autre que 1,2,3,4,6.
Pour sortir de Jouer,Entrez un mode,Entrer un pseudo puis entrer quitter.
